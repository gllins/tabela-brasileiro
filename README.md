# Tabela Brasileirão Interativa

Projeto web simples que simula a tabela do Campeonato Brasileiro, permitindo registrar resultados de partidas e atualizar automaticamente a classificação dos times.

## Funcionalidades

- Exibição da tabela com:
  - Pontos (PTS)
  - Jogos (J)
  - Vitórias (V)
  - Empates (E)
  - Derrotas (D)
  - Gols Pró (GP)
  - Gols Contra (GC)
  - Saldo de Gols (SG)

- Classificação automática dos times baseada nos resultados

- Destaque visual por cores:
  - 🔵 Libertadores (top 4)
  - 🟠 Qualificatórias
  - 🟢 Sudamericana
  - 🔴 Rebaixamento (últimos 4)

- Painel de registro de partidas:
  - Seleção de dois times
  - Inserção de gols
  - Atualização instantânea da tabela

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

> Todo o código foi desenvolvido em um único arquivo HTML, incluindo estrutura, estilo e lógica.

## Aprendizados

- Manipulação de dados em JavaScript
- Atualização dinâmica do DOM
- Lógica de classificação esportiva
- Organização de interface interativa

## Limitações

- Não possui persistência de dados (dados são perdidos ao recarregar a página)
- Estrutura concentrada em um único arquivo (HTML, CSS e JS juntos)
- Projeto focado em lógica, não em arquitetura escalável

## Como executar

1. Baixe ou clone o repositório
2. Abra o arquivo `.html` em qualquer navegador
- Implementar rodadas completas do campeonato
